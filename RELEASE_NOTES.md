# AWS Tunnel Manager - Release Notes

## Versão 1.0.0

### Funcionalidades

- Gerenciamento de perfis de conexão SSH tunnel para AWS EC2
- Interface gráfica intuitiva para configurar e gerenciar tunnels
- Monitoramento de estado de conexão em tempo real
- Ícone de bandeja com indicador visual de conexão
- Suporte para múltiplas regiões AWS
- Configuração de portas local e remota personalizáveis

### Sistema

- Ícone do aplicativo otimizado para Linux (256x256)
- Pacote .deb para instalação no Ubuntu/Debian
- Suporte a AppImage para distribuição universal
